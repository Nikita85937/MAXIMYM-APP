# 🚀 БЫСТРЫЙ СТАРТ

## 1. Установите:
```bash
flutter pub get
```

## 2. Запустите:
```bash
flutter run
```

## 🎬 При запуске:

1. **Экран приветствия** ✈️
   - Самолет и облака
   - Кнопка "НАЧАТЬ"

2. **Разрешения** 📋
   - Контакты, микрофон, камера
   - Кнопка "РАЗРЕШИТЬ"

3. **Главный экран** 💬
   - Чаты готовы!
